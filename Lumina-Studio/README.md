🎨 EASY CUSTOMIZATION (CSS Variables)
----------------------------------------------------------------------
Lumina is designed with scalability in mind. You can change the entire 
color scheme of the website in less than 5 seconds!

Open "css/style.css" and locate the global variables at the top:

:root {
  --bg-primary: #0a0a0c;      /* Main background color */
  --accent-color: #00d4ff;    /* Vibrant Cyan accent color */
  --text-main: #ffffff;       /* Primary text color */
}

Simply change the HEX color codes in the ':root' block, and the entire 
template will update instantly across all components.
